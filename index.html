Example or Exercise:

Example 1 :

console.log(nm);

let nm = "name";

Answer or Error: ReferenceError

Why: let variable declaration se pehle access nahi hota (TDZ).

Example 2 :

console.log(b);

var b = "username";

Answer or Error: undefined

Why: var hoisting hota hai, default value undefined hoti hai.

Example 3 :

var x = 1;

{ var x = 2; }

console.log(x);

Answer or Error: 2

Why: var block scoped nahi hota.

Example 4 :

let a = 10;

{ let a = 20; console.log("Inside:", a); }

console.log("Outside:", a);

Answer or Error: Inside 20, Outside 10

Why: let block scoped hota hai.

Example 5 :

const person = {name: "demo"};

person.name = "user"; ✅

person = {}; ❌

Answer or Error: Property change allowed, reassignment not allowed

Why: const reference fix hota hai, object ke properties change ho sakte hain.
